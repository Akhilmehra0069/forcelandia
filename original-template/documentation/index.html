"Expand" class

Expand class in the slider title

You can set the background of the title in slider to be expanded or to be the same width as a title. To expand it (like on demo website) include expand class near the title class

<div class="title expand move""><h1">Web Design 3.0</h1"></div">

Expand class for expanding blocks of content

You can also expand background color of the content to the right or left. To achieve that:

  • Include expand class near the block class
  • Then enter left or right class in the same place

    So for example if you want to expand #testimonials section to the right:
    <section id="testimonials" class="block expand right background-color-grey-light">

Map setup

Set the center of map

  • Find out latitude and longitude of your desired map center
  • Edit _latitude and _longitude variables in the footer by new values

Edit the address in the map marker

  • Find markerAddress variable
  • Edit it in this way: "Your address" - you need to keep the quotes!
  • If you want to add new line, add <br> tag.
    For example: "16-22 Great Russell St,<br> Los Angeles, United States"

Universal classes

You can use some of the universal classes in the HTML, so you don't need to create CSS style for every element

  • background-color-grey-light - set the background color to light grey
  • background-color-grey-dark - set the background color to dark grey
  • background-color-default - set the background color to the one you have selected as default
